FAQ

CLI 없이 API만으로 가능한가요?

네. 모든 기능은 REST API 기반으로 설계되어 자동화와 서버 환경에 최적화되어 있습니다.

장시간 테스트 시 메모리는 안전한가요?

요청 단위 데이터를 저장하지 않고 히스토그램 기반 집계를 사용해 메모리를 O(1)로 유지합니다.

내부망이나 사설 IP 테스트도 되나요?

기본 정책상 차단됩니다. 이는 보안 및 오남용 방지를 위한 설계입니다.

실시간 메트릭은 어떤 단위로 제공되나요?

기본적으로 1초 단위 스냅샷으로 RPS, 실패율, p50/p95/p99 지연 시간을 제공합니다.
